HTML

Hypertext Markup Language (HTML)is the standard Language for creatingwebpages and web appication. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describe the structure of a web page sematically and originally included cues for the apprearnce of the document.

Html elements are the building blocks of HTML pages with HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embeded into the renderd page. It provides a means to create structured documents by denoting structual sematics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle bracket.
